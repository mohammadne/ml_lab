1. install uv:
macos/linux:
curl -LsSf https://astral.sh/uv/install.sh | sh

windows:
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"

2. create a project in pycharm; env: choose uv
3. note the path where uv is installed: /Users/sid/.local/bin/uv

4. Install the dependencies
uv add "mcp[cli]" httpx python-dotenv requests

5. create mcp server code

6. install claude desktop

7. create claude_desktop_config.json

nano ~/Library/Application\ Support/Claude/claude_desktop_config.json

8. Paste this content in the json file (update uv path and directory path)

{
  "mcpServers": {
    "weather": {
      "command": "/Users/sid/.local/bin/uv",
      "args": [
        "--directory",
        "/Volumes/MyDrive/3_udemy/3_udemy_src/2_rec_code/9_mcp/9_2_build_an_mcp_server",
        "run",
        "weather.py"
      ]
    }
  }
}
